# magazine-site

### 實作網址： https://judy-nihao.github.io/magazine-site/

### 實作功能
- 背景換色：滾動時背景會隨雜誌配色切換
- 滾動吸附：電腦版畫面滾動時，畫面會逐一吸附各個雜誌區塊
- 目錄提示：滾動時，目錄名稱會對應雜誌編號加粗
- RWD自適應佈局：行動版版面調整，以及目錄改為可收闔式
